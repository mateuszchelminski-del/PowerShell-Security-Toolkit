# PowerShell Security Toolkit

## Overview
**PowerShell Security Toolkit** is a curated collection of **defensive, read-only PowerShell scripts** designed for Windows security auditing, endpoint visibility, threat hunting, and incident response.

This repository demonstrates hands-on knowledge of **Windows internals, PowerShell automation, and Blue Team security operations**, making it suitable for SOC analysts, security engineers, and IT auditors.

All scripts are **safe for lab and enterprise environments** and do **not modify system state**.
