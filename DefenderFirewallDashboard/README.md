# Defender & Firewall Dashboard

## Description
Provides a security posture overview of Microsoft Defender and Windows Firewall configurations.

## Collects
- Microsoft Defender real-time protection status
- Defender signature and engine versions
- Firewall profile states
- Enabled inbound allow rules
- Attack Surface Reduction (ASR) rules (if available)

## Usage
```powershell
.\DefenderFirewallDashboard.ps1
